Job Fit Scorer — Settings

All scoring happens on the job-store backend. The backend holds the resume, the API key, the model, and the prompt — the plugin just sends job descriptions and renders responses.

URL of the job-store Flask backend (e.g. http://127.0.0.1:5000 for local dev, https://jobs.example.com for hosted).

Localhost URLs work out of the box. Any other host triggers a one-time Firefox permission prompt the first time you Save or Test — accept it to let the extension reach that host.

Resume edits happen on the backend host (resume_details.yaml in the job-store working directory) — not here. Career-growth keywords are configured via the GROWTH_KEYWORDS env var on the backend.
